Solicitud para la Busqueda de Jugadores

En esta página se generó un CRUD para las siguientes rutas:

  • Usuarios(/usuarios)
  • Deportes(/deportes)
  • Falta Uno(/faltauno)

La temática elegida es las creacion de solicitudes para buscar jugadores de distintos deportes. La idea es que yo, por ejemplo, reservé una cancha de futbol, tenis, o paddle, etc, y me falta alguna persona para completar el equipo. Entonces recurro a generar una solicitud donde le envio varios parametros y asi lo publico; En este caso, en una base de datos.
